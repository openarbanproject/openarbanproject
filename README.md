## Welcome to the openArbanProject!

The purpose of oAP is to create music scores of public domain music using Lilypond.
The project was created and continues to be managed by Rich Colquhoun, who is a cornet/trumpet player from the UK. 🎺

As indicated by the name, the original goal and current primary purpose of oAP is to create a version of the Arban Cornet Method that is free to use, edit and redistribute without copyright infringement. This empowers musicians and students to use the material for learning in ways beyond simply playing an exercise:
* ✏ Create multiple versions of an exercise with different articulations
* ✏ Reinterpret a study using up-to-date style
* ✏ Create an authentic 19th Century version for performance practice practise
* ✏ Transpose a study into different keys or tonalities
* 🤔 etc.

As you can see there are examples of other public domain brass music that have their own repositories, including a version of the Haydn Trumpet Concerto for students to mark-up as they choose.
There is a disscussion tab in the openArbanBook repo that you're welcome to post in and help us to create a community around this project.

Thanks for visiting. 👋
